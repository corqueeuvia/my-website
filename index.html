Materiais Lovelace

A segunda tarefa foi cumprir uns simples requisitos. Nessa eu exagerei!

Criar uma página HTML que possua:

  • Formatação de textos (ex.: "b", "strong", "i");
  • Adição de cores usando o atributo "style";
  • Textos clicáveis que direcionem para outra página usando a tag "a";
  • Imagens que, ao ser clicadas, abram a imagem em uma nova aba usando o atributo "target";
  • Uma tabela com no mínimo 4 linhas e 3 colunas usando a tag "table";
  • Um comentário que não será exibido ao usuário.

Em minha defesa: não se dá o fim de semana todo como prazo se não quer receber menos de 600 linhas de código como resposta!

Radio button

Aqui aprendemos a usar o <input type="radio"> e eu o apliquei em uma questão que você precisa se perguntar.

Requisitos:

  • Usar o input do tipo radio;
  • Ao clicar no label do botão a seleção deve ser feita;
  • As opções devem ser excludentes entre si: ao clicar em uma, a anteriormente selecionada deve ser desmarcada.
